# Login-System
Login / registration system that records user information like username, email, password and last login in an SQL database (IBM Db2) on cloud.

## How to use it?
First, you have to create an account in IBM Cloud and choose the IBM Db2 service.
After that, you will be able to get your credentials and change the lines 10 and 11 with your ID and password, respectivily.

## Creating a table 
The project only uses one table called users, so if you want to create with same columns and data types, run the <a href="https://github.com/mateusvictor/Login-System/blob/main/create_table.sql">create_table.sql</a> file in the console of your database.
